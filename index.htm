FEATURES:
- Visual Studio 2008 / C#
- SQLite databases
- photo search using hierarchical categories
- locating moved photos based on filename, filesize and md5 hash
- read-only mode
- possibility to hide photos


INSTALLATION
1) Download System.Data.SQLite.dll and the latest version of Photo.org.exe and place them in the directory of your choice.
2) That's all

Some "instructions":

COMMAND LINE PARAMETERS
- Use read-only mode (-r, -R)
- Show hidden photos (-h, -H)

CATEGORY TREE
- Move category node (drag/drop w/ Shift)

THUMBNAIL VIEW MODE
- Open category dialog (Space)
- Open filter dialog (Ctrl+F)
- Remove selected photos from database (Delete)
- Switch to photo view mode (Double click a thumbnail, Enter)
- Hide photos (Ctrl+H)
- Unhide photos (Ctrl+Shift+H)
- File info (F1)
- Rename file (F2)

PHOTO VIEW MODE
- Next photo (mouse wheel down, down arrow, right arrow, page down, N)
- Previous photo (mouse wheel up, up arrow, left arrow. page up, P)
- Return to thumbnail view (Esc, Enter)
- Assign category (drag/drop to category tree node)
- Remove category assignment (drag/drop to category tree node w/ Shift)
- Open category dialog (Space)
- Assign last used category to current photo (+)
- Remove last used category assignment from current photo (-)
- Begin slideshow (F8)
- End slideshow (Esc)
- Hide photo (Ctrl+H)
- Unhide photo (Ctrl+Shift+H)
- File info (F1)
